# АИС "Ведение оптовых заказов" (АИС ВОЗ)

Информационная система для компании по оптовой продаже товаров.
Платформа: WPF (.NET).

## Основные функции
- Ведение справочника товаров
- Ведение базы заказчиков
- Оформление и хранение документов по сделкам
- Отслеживание финансовых показателей

## Стек технологий
- WPF, C#, .NET
- NS SQL Server (или SQLite)
